Narrative Design

This page will serve as a live portfolio showcasing my work with HTML, CSS and JavaScript.

I'd like to start with a personal shout-out to Tom and Quinn, who have provided resources, guidance and troubleshoot assistance, and who were overall instrumental in getting me started!

So far in my journey I've learned HTML, CSS, JavaScript, some Python and SQL, and I've begun familiarising myself with CMS platforms, using resources such as W3Schools, freeCodeCamp, and G4G primarily. For the most part, I find myself gravitating toward front-end, I enjoy the more immediately visible quality and the creative side of things here.

Beginning with raw CSS code, I moved on to Bootstrap and wound up incorporating a number of libraries. Of course, I first had to learn how to work with back-end programs like NPM and Python.

Getting to play around with these libraries proved initially quite challenging, as it felt like having to entirely relearn how to code styling elements. However, proficiency with responsive design was a focus of mine, and ultimately found this a lot smoother with BS -- once I figured out the trick to it.

Other resources used were: Animate On Scroll, Haikei, GitHub and Figma as my primary resources, from learning to creating vectors. It's been a rewarding and enriching journey, not without its humbling frustrations. I've become totally fixated on the entire creative process, and hope that this will be my first site of many.
